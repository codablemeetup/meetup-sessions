# AI Assisted Software Engineering

**Date:** 14 March 2026
**Location:** The Zone, Gbagada, Lagos
**Event page:** [Luma Calendar](https://luma.com/codablemeetup)

## About

This event explored how AI is reshaping the way developers write, review, and ship code — from copilots and code generation to testing, debugging, and architecture decisions.
